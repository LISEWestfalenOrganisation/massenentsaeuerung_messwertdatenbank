Einzelblatt-Entsäuerung:

Das Bückeburger-Verfahren, inzwischen mehrfach modifiziert und optimiert, ist ein wässriges Entsäuerungsverfahren, das exklusiv von der Fa. Proarchive Systems Oliver Zinn angeboten wird. 
Die Entsäuerung funktioniert durch das Abfangen freier Säureionen durch Magnesiumhydrogencarbonate im wässrigen Medium, und das Anlegen einer alkalischen Reserve durch einen Überschuss der Carbonate. Ein Auslaufen von Schreibstoffen wird durch die Zugabe von Fixiermitteln verhindert, eine Zugabe von Methylcellulose bewirkt eine Nachleimung der Papiere.
