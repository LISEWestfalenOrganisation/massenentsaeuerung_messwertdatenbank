Blockentsäuerung:

Die Blockentsäuerung beschreibt ein standardisiertes Verfahren der Mengenentsäuerung, bei dem Archiv- und Bibliotheksgut in einer nichtwässrigen Lösung entsäuert werden. Hierbei wird gleichfalls eine alkalische Reserve eingebracht. Es handelt sich um nichtwässrige Flüssigverfahren, die entweder heterogen mit dispergierten Partikeln (Bookkeeper®, ZFB:2-Verfahren) oder homogen auf Lösungsmittelbasis (PAL Book Saver®, papersave®) arbeiten.
